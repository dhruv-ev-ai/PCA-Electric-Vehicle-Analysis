⚡ Green Skills & AI Projects

This repository contains projects completed during the Green Skills and Artificial Intelligence Course.
The goal of these projects is to apply machine learning techniques to real-world datasets and understand concepts like data preprocessing, dimensionality reduction, and model building.

---

📊 Project: Principal Component Analysis (PCA)

This project demonstrates the use of Principal Component Analysis (PCA) to reduce the dimensionality of an Electric Vehicle Energy Consumption dataset.

🎯 Objectives

- Understand the concept of dimensionality reduction
- Apply PCA using Scikit-Learn
- Visualize variance explained by principal components
- Reduce dataset complexity while preserving important information

---

🛠 Technologies Used

- 🐍 Python
- 📊 Pandas
- 🔢 NumPy
- 🤖 Scikit-Learn
- 📈 Matplotlib

---

🚀 Key Concepts Covered

- Data preprocessing
- Feature scaling
- Principal Component Analysis (PCA)
- Explained variance analysis
- Data visualization

---

📚 Learning Outcome

Through this project, I learned how PCA helps simplify complex datasets while retaining most of the important information, which is useful for machine learning and data analysis tasks.
